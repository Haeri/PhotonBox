# PhotonBox
A modern OpenGL Game-Engine with focus on PBR

## Rendering 
- Forward Rendering
- Point lights, Directional lights
- SkyBox/Enviroment Mapping
- Blinn-Phong BRDF shader with Fernel reflectance
  - Diffuse Map
  - Normal Map
  - Specular Map
  - Ambient occlusion Map
  - Emission Map

## Systems
- Component system
- Scene loading
- Transformation system
- Generic Materials
- Input
